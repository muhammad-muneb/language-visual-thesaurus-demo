# Language Visual Thesaurus

A dynamic vocabulary learning tool using graph-based word relationships and intuitive mind maps.

## 🎥 Demo Video
[![Watch Demo](https://img.youtube.com/vi/iItR0_uNECk/0.jpg)](https://youtu.be/iItR0_uNECk)

## 🛠️ Tech Stack
- Frontend: React JS
- Backend: Node js
- Database: MySQL

## ✨ Features
- Interactive mind map for word associations
- Dynamic word relationship visualization
- User-friendly interface
- Frontend and backend integration
- Data visualization
